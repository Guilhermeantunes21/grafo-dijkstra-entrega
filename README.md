# Algoritmo de Dijkstra em Python

Este projeto demonstra a utilização do algoritmo de Dijkstra para encontrar a menor distância entre pontos de um grafo.

## Cenário

O exemplo simula um sistema de rotas para entrega de comida.

Locais:
- Restaurante
- Rua A
- Rua B
- Praça
- Cliente

Cada conexão possui uma distância em quilômetros.

## Tecnologias

- Python
- Algoritmo de Grafos
- Dijkstra

## Como executar

1. Instale Python
2. Execute:

python main.py

## Exemplo de saída

Menor distância: 2 km
